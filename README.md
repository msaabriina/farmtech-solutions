# 🌱 FarmTech Solutions

## 📌 Descrição do Projeto
A FarmTech Solutions é uma aplicação desenvolvida com o objetivo de simular uma solução digital para o setor agrícola, permitindo o gerenciamento de lavouras, cálculo de áreas e estimativa de insumos de forma simples e eficiente.

---

## 🌍 Visão Geral da Solução
O sistema foi desenvolvido para auxiliar produtores rurais no planejamento de suas plantações, automatizando cálculos importantes e organizando informações de forma estruturada.

A solução permite:
- Cadastro de lavouras  
- Cálculo de área (metros quadrados)  
- Estimativa de insumos agrícolas  
- Armazenamento dos dados em arquivo CSV  
- Análise estatística utilizando R  
- Consulta de dados climáticos via API  

---

## 🐍 Aplicação em Python

A aplicação principal foi desenvolvida em Python e executada no terminal.

Funcionalidades:
- Menu interativo  
- Cadastro de culturas (Café e Cana-de-açúcar)  
- Cálculo de área (comprimento x largura)  
- Cálculo de insumos:  
  - Café → fertilizante fosfatado (0,25 kg por m²)  
  - Cana → herbicida (0,5 litros por rua)  
- Atualização e exclusão de registros  
- Exportação dos dados para arquivo CSV
